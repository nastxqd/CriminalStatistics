# Веб-приложение для визуализации аналитической информации об уровне преступности в регионах России

**Автор:** Андриянова Анастасия Андреевна  
**Дата завершения:** 22 января 2025 г.

---

## 📌 Цель проекта

Создание веб-приложения, предоставляющего пользователю доступ к аналитической информации о преступности в различных регионах России с возможностью интерактивной визуализации данных.

### Задачи проекта:
- Разработка удобного интерфейса для взаимодействия с данными
- Обработка и подготовка исходных статистических данных
- Реализация функционала фильтрации и сортировки по регионам
- Внедрение алгоритмов визуализации данных в виде графиков
- Обеспечение корректного отображения на мобильных устройствах

---

## 📁 Данные

**Источник:** Официальные открытые данные Министерства внутренних дел Российской Федерации (https://мвд.рф/открытые-данные)

**Структура таблицы Statistics:**

| Поле | Описание |
|------|----------|
| ID | Уникальный идентификатор записи |
| Subject | Субъект (регион) Российской Федерации |
| Point | Пункт ФПСР |
| Factor | Наименование статистического показателя |
| Amount | Значение статистического показателя (январь-декабрь 2024 г.) |
| Status | Статус стадии рассмотрения преступления |

**Категории статусов:**
- Зарегистрированные преступления
- Расследованные преступления

## ⚙️ Функциональные возможности

### Основной функционал:
- ✅ Фильтрация данных по регионам РФ
- ✅ Визуализация статистики преступлений в виде круговых диаграмм
- ✅ Сравнение зарегистрированных и расследованных преступлений

### Пользовательские сценарии:

   - Зарегистрированные преступления
   - Расследованные преступления
3. **Сравнительный анализ** — возможность визуального сравнения категорий преступлений в выбранном регионе

## 🔧 Процедура обработки данных

### 1. Группировка преступлений по регионам
- Данные группируются по полю `subject` (регион)
- Используется `Map<String, List<Criminal>>` для хранения сгруппированных данных

### 2. Сортировка регионов
- Регионы сортируются по алфавиту для удобства навигации
- Сортировка регистронезависимая (первая буква приводится к нижнему регистру)

### 3. Разделение по статусу
- `processReg()` — выделение зарегистрированных преступлений
- `processDone()` — выделение расследованных преступлений
- Используется Stream API с `Collectors.groupingBy()`

### 4. Подготовка данных для визуализации
- Формирование меток (категории преступлений)
- Сбор числовых значений (количество)
- Генерация PNG-изображения диаграммы
- Кодирование в Base64 для встраивания в HTML

## 🚀 Запуск приложения

### Требования
- Java 8 или выше
- MySQL Server
- Maven

### Инструкция по запуску

1. **Клонирование репозитория**
```bash
git clone <ссылка на репозиторий>
cd criminal-web-app
```

2. **Настройка базы данных MySQL**
```sql
CREATE DATABASE criminal_statistics;
-- Импорт данных из предоставленного дампа
```

3. **Настройка подключения** (`application.properties`)
```properties
spring.datasource.url=jdbc:mysql://localhost:3306/criminal_statistics
spring.datasource.username=your_username
spring.datasource.password=your_password
```

4. **Сборка и запуск**
```bash
mvn clean install
mvn spring-boot:run
```

5. **Открытие в браузере**
```
http://localhost:8080
```
